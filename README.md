### Greedy 알고리즘(탐욕 알고리즘)

### 에라토스테네스의 체(소수찾기)
소수의 배수는 소수가 아닌 것을 이용, 마지막 수의 제곱근까지만 확인하여 모든 수의 소수 확인 가능  
ex)소수 찾기
